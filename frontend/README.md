# BuildTrack Frontend

This folder contains the Angular frontend for BuildTrack.

## Requirements

- Node.js 20.19 or newer is recommended.
- npm is required.

## Install Dependencies

From the project root:

```powershell
cd frontend
npm install
```

## Run the Frontend

Start the development server:

```powershell
cd frontend
npm start
```

Open:

```text
http://127.0.0.1:4200
```

## Other Useful Commands

Build for production:

```powershell
npm run build
```

Run tests:

```powershell
npm test
```
